Shortcuts and Controls

ITEMS
  • create item ctrl + click / doubleclick
  • draw items shift + ctrl + mousemove
  • delete item mmb on item
  • erase items alt + shift + mousemove
    • SELECTIONS
  • select items mousedrag
  • add item shift + click
  • add items shift + mousedrag
  • select items on row alt + x (mouse row)
  • move selected x ctrl + x, return to confirm, esc to abort
  • move selected y ctrl + y, return to confirm, esc to abort
  • duplicate selected ctrl + d
  • delete selected delete / backspace
  • clear all press x button in main control
SNAPSHOTS
  • create snapshot alt + s
  • recall snapshot ctrl + 0-9
  • snapshots are for fast editing, they will get lost on closing the sequencer window. to save a snapshot, save a new preset in pd
NAVIGATION
  • scale x-axis wheel
  • scale y-axis ctrl + wheel
  • scroll y-axis shift + wheel
PLAYER
  • start/stop playing space
  • set transport start click into x-scale
  • set loop-region start ctrl click into x-scale
  • sets loop-region end alt click into x-scale
  • loop-regions are draggable
PARAMETERS
  • select parameter/create new click name on the left
  • select parameter ctrl + up- or downArrow
  • set item(s) at current position click
  • delete parameter on item mmb
  • draw values ctrl + lmb
  • resize parameters hight ctrl + mousemove on bottom left edge
  • using solo/mute will show the parameters of the soloed/not muted items only
SOLO/MUTE
  • exclusive solo ctrl + click solo
  • unmute/unsolo all ctrl + s
VARIOUS
  • show/hide main control ctrl + m
  • log received data ctrl + l
  • log sent data ctrl + r
MIDI
  • imports midi files of type 1
